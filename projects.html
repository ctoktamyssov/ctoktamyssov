Three Body Problem

Simulates the classical three-body gravitational problem in real time using C++ and OpenGL, with RK4 numerical integration for stable physics and GLM for vector math. It visualizes chaotic orbital motion.

Card Detection

Detects playing cards in real-time using a webcam, using YOLOv8 for object detection and OpenCV for video capture and visualization. It is designed for poker applications where you want live recognition of cards.
